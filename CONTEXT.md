# IW

IW 管理用户日常生活数据及围绕这些数据产生的辅助能力。本 glossary 统一各业务域在设计与实现中使用的领域语言。

## Language

### 衣橱

**衣物源图**:
用户为衣物保存的原始图片，其 URL 稳定标识不可变的图片内容。
_Avoid_: 当前展示图、衣物封面图

**衣物优化图**:
由衣物图片优化任务生成并成功关联到衣物的派生图片；删除衣物优化图不会改变衣物源图或历史任务尝试。
_Avoid_: 衣物源图、当前展示图、AI 图片

**衣物图片优化任务**:
一次针对特定衣物及其衣物源图发起、具有稳定身份并可追踪至终态的图片优化过程。
_Avoid_: 图片优化请求、图片生成记录、临时任务

**任务尝试**:
衣物图片优化任务的一次实际执行；同一任务可因用户主动重试产生多次任务尝试，但任务身份保持不变。
_Avoid_: 重试任务、新任务
